# 第二章「深淵古神」— 待使用者拍板的難度/UX 抉擇（2026-07-05）

## TL;DR
第二章**技術上已就緒**：接通了「莎菈(章節boss·全滿血) → 深淵古神(最終boss)」，而且**當初讓它 unshippable 的 soft-lock 已徹底根治**（decay + 復活 + 防軟鎖 → 實測 0 stall）。
**但**：完整 2 章在 meta0（無全局成長）太殘酷。這是**難度/UX 抉擇**，留給使用者決定，暫時維持單章。

## 實測數據（動腦 adapter · 20 seed）
| | 單章(現行) | 2章(深淵古神) |
|---|---|---|
| meta0 各組合 | 30–65% | **0–15%**(太殘酷) |
| meta中 | 50–85% | 20–60% |
| meta滿 | 95–100% | **85–90%**(可通關) |
| STALL | 0 | **0**(soft-lock 已解) |

→ 2 章是可行的「meta 解鎖終盤」，但新玩家 meta0 幾乎全敗（單章版有 30–65% 的爽快早期勝利）。

## 選項（請使用者選）
- **A. 維持單章**（現行）：meta0 30–65% 有成就感。深淵古神當彩蛋/日後用。
- **B. 2 章常駐硬核**：內容量翻倍、但新玩家前期幾乎全敗、靠全局成長解鎖通關（爬塔式 ascension 體驗）。
- **C. 2 章但調軟**：縮短 ch2 或降深淵古神數值，讓 meta0 拉到 ~20–30%（半苦半爽）。
- **D. ch2 當「進階解鎖」**：首次通關莎菈後才開放第二章（新手先爽單章、老手挑戰 ascension）。→ 需加解鎖旗標(存 meta)。

## 魚乾建議
傾向 **D**（最兼顧新手成就感 + 老手深度）或 **C**。B 對第一印象太傷。
一鍵重接 ch2 的 diff 已知（MAP 裡莎菈改 non-final + append `crabgen/campfire/murk/shop/abyssking`），使用者定案後 5 分鐘可上。

## 已完成的技術面（不浪費）
- 深淵古神(abyssking hp240·抗火抗冰·怕雷·召喚+流血+橫掃+高smash) 已在 ENEMIES 定義好。
- 章節 boss 轉場（非 final boss → 全體回滿 → 續下一章）機制在 winBattle 已就緒、實測正常。
- soft-lock 根治：round>15 敵人遞增崩解 decay + 戰後復活 + 單回合出牌上限 → 任何局保證終止（這是本次最大收穫、單章也受惠）。
